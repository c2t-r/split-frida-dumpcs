# split frida dumpcs

split dump.cs generated by `frida-il2cpp-bridge` into multiple files  
this tool helps to manage dump.cs by git

## usage

```bash
uv run main.py <dump.cs> [output_dir]
uv run main.py "../dump.cs" "dump"
```
